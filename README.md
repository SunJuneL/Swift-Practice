## 계산기 앱 만들기 - MyCalculator
### 아래 내용들을 포함하는 앱 만들기
1. 숫자 패드로 누르면 해당 숫자가 화면에 나옴.
2. 연산자를 누르기 전까지 숫자 패드를 누르면 숫자가 뒤에서 추가됨. (ex. 1 -> 12)
3. 연산자를 누르면 계산 결과 혹은 피연산자가 나옴.
4. 실제 계산 순서가 계산기 앱에 적용됨. (곱하기 & 나누기 -> 더하기 & 빼기)
5. 괄호 패드는 일단 존재하지 않고, 괄호는 수식에 포함되지 않음.
